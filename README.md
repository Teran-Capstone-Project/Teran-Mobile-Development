# Teran: Tenang Pikiran

**Deskripsi:**
Teran: Tenang Pikiran adalah aplikasi kesehatan mental yang membantu pengguna menjaga kesehatan pikiran melalui berbagai fitur seperti catatan jurnal, meditasi, dan survey stress berbasis machine learning. Pengguna juga dapat melakukan posting curhat baik secara anonim maupun non-anonim menggunakan cloud API. Aplikasi ini dikembangkan menggunakan bahasa Kotlin dengan XML sebagai interface, sebagai bagian dari proyek tugas akhir Bangkit yang berkolaborasi dengan bidang machine learning, mobile development, dan cloud computing.

## Fitur Utama

- **Catatan Jurnal**: Mencatat perasaan dan pikiran harian pengguna.
- **Meditasi**: Panduan meditasi untuk membantu pengguna mencapai ketenangan pikiran.
- **Survey Stress**: Menggunakan machine learning untuk mengevaluasi tingkat stress pengguna.
- **Posting Curhat**: Pengguna dapat membagikan cerita atau curhatan baik secara anonim maupun non-anonim melalui cloud API.

## Teknologi yang Digunakan

- **Bahasa Pemrograman**: Kotlin
- **Interface**: XML
- **Machine Learning**: Digunakan untuk evaluasi tingkat stress
- **Cloud Computing**: Untuk mendukung fitur posting curhat
